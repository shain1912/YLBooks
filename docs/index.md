# 바이브코딩으로 만드는 내 키링 게임기

**부제: AI랑 같이 펌웨어 짜고, 내가 만든 키링으로 리듬게임까지 — 22장 DIY 코딩 교재**

이 책은 한국 대학생이 AI 페어 코딩(Claude Code · Antigravity)을 도구로 삼아, 교수님이 주신 PCB 키링 보드에 펌웨어를 올리고 그 키링으로 직접 만든 리듬게임과 조이스틱 슈팅 게임을 플레이할 수 있게 되기까지의 22단계를 따라가는 DIY 교재입니다.

## 다섯 부 구조

- **1부 — 바이브코딩 첫걸음** (1~3장) — Claude Code / Antigravity 설치, 첫 프롬프트
- **2부 — 키링이 살아나는 순간** (4~8장) — PCB 인식부터 OSU 4버튼 컨트롤러 완성
- **3부 — 내 손으로 만드는 리듬게임** (9~15장) — pygame으로 노트·차트·판정·캘리브레이션·발표 시연
- **4부 — 조이스틱으로 확장** (16~19장) — 게임패드 모드 + 우주선 슈팅 게임
- **5부 — 마무리와 자랑** (20~22장) — 케이스, 깃허브 공개, 발표 슬라이드

## 약속

마지막 페이지를 덮을 때 손에 쥐는 것:
